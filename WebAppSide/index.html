📖 使い方ガイド

🚀 クイックスタート

Step 1: BLE接続

  • [Connect Device] ボタンをクリック
  • デバイスリストから ESP32-S3-MICON を選択
  • 接続完了を確認（ステータスが「Connected」に変わる）

Step 2: ファームウェアアップロード

  • [Choose File] で .bin ファイルを選択（最大2MB）
  • [Upload Firmware] をクリック
  • 進捗バーで進行状況を確認
  • 完了後、デバイスが自動的に再起動

Step 3: デバッグモニタ

  • [Subscribe] をクリック
  • ESP32からのログがリアルタイムで表示されます
  • デフォルトで「Hello World via BLE」が1秒ごとに送信されます

Step 4: Wi-Fi設定（オプション）

  • Wi-Fi機能を使う場合のみ設定
  • SSIDとパスワードを入力
  • [Send] をクリック
  • デバイスがWi-Fiに接続（IPアドレスが表示される）

⚠️ 注意事項

  • ファームウェア更新は 完全にBLE経由 で実施されます
  • アップロード中はデバイスとの距離を近く保ってください
  • ファイルサイズは2MB以下である必要があります
  • Safari は Web Bluetooth API 非対応です（Bluefy推奨）
